# 下一步建议

考虑预算有限，建议按这个顺序继续，不要一口气铺太大。

## 最高优先级

1. 把左侧剪辑区做得更像剪映：预览、文稿、时间线、属性面板四块固定下来。
2. 做片段级时间线操作：选中、删除、恢复、拖动 trim 起止点。
3. 让字幕编辑更明确：保存状态、失败提示、正在重渲染提示、渲染完成提示。
4. 把 Agent 的每次修改变成可审阅列表：用户能看见“它准备删哪里、为什么删、能不能撤回”。

## 第二优先级

1. 完善智能精剪：停顿、口癖、重复内容、低信息密度段落。
2. 完善字幕样式：字号、位置、描边、关键词高亮、模板。
3. 补素材/B-roll 工作流：推荐、替换、启用/禁用。
4. 做导出 preset：抖音、小红书、B站、YouTube Shorts。

## 暂缓

- 快捷键：用户明确说先不做。
- 大规模重构：除非会阻碍功能闭环。
- 专业 XML/FCPXML/OTIO：等基础剪辑体验稳定后做。
- 真 AI 视频生成：成本较高，先用可替换素材槽位跑通流程。
